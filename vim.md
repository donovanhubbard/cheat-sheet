# vim cheat sheet

# Page Up/Down
| Mode | Key    | Command                |
| ---- | ------ | ---------------------- |
| n    | ctrl+d | Scroll down half page  |
| n    | ctrl+u | Scroll up half page    |
| n    | ctrl+f | Scroll down whole page |
| n    | ctrl+b | Scroll up whole page   |
| n    | ctrl+e | Scroll down 1 line     |
| n    | ctrl+y | scroll down 1 line     |

# Search
| Mode | Key | Command          |
| ---- | --- | ---------------- |
| n    | /   | Search forward   |
| n    | ?   | Search backwards |
