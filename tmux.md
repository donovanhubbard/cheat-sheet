# tmux

## Sessions
| key | alias | action               |
| --- | ----- | -------------------- |
| `s` |       | View all sessions    |
| `(` |       | Next session         |
| `)` |       | Previous session     |
| `$` |       | Rename session       |
|     | `u`   | Create new session   |
|     | `e`   | Kill session by name |

## Popups
| key | alias | action           |
| --- | ----- | ---------------- |
|     | `y`   | blank shell      |
|     | `t`   | tmux cheat sheet |
|     | `v`   | vim cheat seet   |

## Copy Mode
| key      | alias | action                  |
| -------- | ----- | ----------------------- |
| `[`      |       | Enter copy mode         |
| `space`  | `v`   | Start selection         |
| `enter`  | `y`   | copy                    |
| `]`      | `P`   | paste                   |
| `ctrl+b` |       | Scroll up full page     |
| `ctrl+f` |       | Scroll down full page   |
| `ctrl+u` |       | Scroll up half page     |
| `ctrl+d` |       | Scroll down half page   |
| `ctrl+y` |       | Scroll up single line   |
| `ctrl+e` |       | Scroll down single line |
| `g`      |       | Goto top                |
| `G`      |       | Goto bottom             |
| `/`      |       | Search forward          |
| `?`      |       | Search backward         |

## Windows
| key | alias | action              |
| --- | ----- | ------------------- |
| `n` |       | Next window         |
| `p` |       | Previous window     |
| `c` |       | Create window       |
| `,` |       | Rename window       |
| `&` |       | Kill current window |

## Panes
| key        | alias | action                                  |
| ---------- | ----- | --------------------------------------- |
| `"`        |       | Horizontal split                        |
| `%`        |       | Vertical Split                          |
| `!`        |       | Convert pane to window                  |
| `z`        |       | Zoom                                    |
| `>`        |       | Pane action menu                        |
| `{`        |       | Swap current pane with previous         |
| `}`        |       | Swap current pane with next             |
| `space`    |       | Cycle through predefined window layouts |
| `option+o` |       | Rotate all panes                        |
